# PowerBI-Sales-Dashboard
# Overview:
This project showcases an interactive Hotel Financial Performance Dashboard built in Microsoft Power BI. The dashboard analyzes hotel financial performance from 2013–2018, providing insights into revenue trends, country performance, hotel performance, and financial metrics through interactive visualizations and dynamic filtering.

The dashboard was designed to transform raw financial data into meaningful business insights, allowing users to explore performance across different years, countries, and hotels.

# Features:
Interactive KPI cards displaying:
Total Value
Average Value
Total Transactions
Hotel Count
Financial trend analysis over time
Country performance comparison
Top-performing hotels by total value
Financial breakdown using a treemap
Interactive slicers for:
Hotel Name
Year
Country
Cross-filtering across all visuals

# Dashboard Insights:
The dashboard helps answer business questions such as:

Which countries generate the highest financial value?
Which hotels contribute the most revenue?
How has financial performance changed over time?
What are the major financial components contributing to total value?
How do financial metrics change when filtering by year or hotel?
